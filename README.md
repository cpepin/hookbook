A collection of helpful React hooks!

## Install
```
npm i --save hookbook
```

## Usage
```
import { useManagedState } from '@cpepin/hookbook';
```

## Available Hooks
* useDebounce
* useManagedState
* useMemoize
* useStateWithGetter

To view documentation on each individual hook, view the corresponding README in the `lib/hooks` folder.
